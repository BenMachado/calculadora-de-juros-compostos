# 📊 Calculadora de Juros Compostos

> Um projeto simples e eficiente para calcular o crescimento do seu patrimônio ao longo do tempo utilizando a fórmula de juros compostos.

## 📝 Descrição

Este projeto é uma ferramenta que permite visualizar como o dinheiro cresce quando aplicado a uma taxa de juros fixa ao longo do tempo (juros sobre juros). Ele foi desenvolvido para facilitar o planejamento financeiro, permitindo simular diferentes cenários de investimento.

## ✨ Funcionalidades

- Cálculo de juros sobre o capital inicial
- Suporte para aporte mensal (investimento regular)
- Simulação de período em anos
- Resultados exibidos no terminal

## 🛠️ Tecnologias Utilizadas

- [Python 3.x](https://www.python.org/)
- [VSCode](https://code.visualstudio.com/)

## 🚀 Como executar

### Pré-requisitos

- Python 3.x instalado no computador

### Passo a passo

```bash
# Clone o repositório
$ git clone https://github.com/BenMachado/calculadora-de-juros-compostos.git

# Entre na pasta do projeto
$ cd calculadora-de-juros-compostos

# Execute o programa
